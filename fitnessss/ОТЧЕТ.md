## 5. КОД СТРАНИЦ И ОСНОВНЫХ МОДУЛЕЙ

### 5.1 Конфигурация и подключение к БД

**Файл: `config/database.php`**

```php
<?php
define('DB_HOST', 'localhost');
define('DB_NAME', 'fitness_center');
define('DB_USER', 'root');
define('DB_PASS', '');

function getDBConnection() {
    static $pdo = null;
    if ($pdo === null) {
        try {
            $dsn = "mysql:host=" . DB_HOST . ";dbname=" . DB_NAME . ";charset=utf8mb4";
            $options = [
                PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION,
                PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
            ];
            $pdo = new PDO($dsn, DB_USER, DB_PASS, $options);
        } catch (PDOException $e) {
            error_log("Database error: " . $e->getMessage());
            die("Ошибка подключения к базе данных");
        }
    }
    return $pdo;
}
?>
```

### 5.2 Функции безопасности

**Файл: `config/config.php`**

```php
<?php
session_start();
require_once __DIR__ . '/database.php';

function isLoggedIn() {
    return isset($_SESSION['user_id']);
}

function isAdmin() {
    return isset($_SESSION['role']) && $_SESSION['role'] === 'admin';
}

function requireLogin() {
    if (!isLoggedIn()) {
        header('Location: login.php');
        exit;
    }
}

function requireAdmin() {
    requireLogin();
    if (!isAdmin()) {
        header('Location: index.php');
        exit;
    }
}

function sanitize($data) {
    return htmlspecialchars(strip_tags(trim($data)));
}
?>
```

## 6. ТЕСТ-КЕЙСЫ И РЕЗУЛЬТАТЫ ТЕСТИРОВАНИЯ

### 6.1 Методики тестирования

Использованы следующие методики:

1. **Черный ящик (Black Box Testing)** - тестирование функциональности без знания внутренней структуры
2. **Белый ящик (White Box Testing)** - тестирование с учетом внутренней структуры кода
3. **Тестирование граничных значений (Boundary Value Testing)** - проверка поведения на граничных значениях
4. **Позитивные тесты** - проверка корректной работы с валидными данными
5. **Негативные тесты** - проверка обработки ошибок и невалидных данных

### 6.2 Таблица тестирования
